# 海龜保育教育與海上安全通報 APP

## 專題簡介
本研究開發一套整合海龜保育教育與海上安全資訊之 Android APP，
提供賞龜熱點地圖、海龜品種介紹、保育法律與檢舉指南、
海龜知識挑戰以及海上求生與救難資訊。

---

## 開發環境
Android Studio

Java

Google Maps API

GBIF API

---

## 系統功能

### 核心一：全台賞龜地圖
### 核心二：海龜品種大點名
### 核心三：保育法律與檢舉指南
### 核心四：海龜守護者挑戰賽
### 核心五：海上求生與救難
