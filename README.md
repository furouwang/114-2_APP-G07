## 海龜保育教育與海上安全通報APP



## 專題簡介

本專題開發一套海龜保育教育與海上安全通報 Android App，
透過 Google Maps API、GBIF 開放資料以及互動式問答功能，
提供海龜保育知識、賞龜熱點資訊、保育法律與海上安全資訊，
提升民眾海洋保育與安全意識。

---

## 開發環境

* Android Studio
* Java
* Google Maps API
* GBIF API
* Android SDK

---

## 系統功能

### 核心一：全台賞龜地圖

* 顯示賞龜熱點位置
* Google Maps Marker 標記
* 預設熱點座標資料

### 核心二：海龜品種大點名

* 串接 GBIF API
* JSON 資料解析
* 顯示海龜品種資訊

### 核心三：保育法律與檢舉指南

* 野生動物保育法
* 檢舉流程說明
* ScrollView 閱讀介面

### 核心四：海龜守護者挑戰賽

* 單選題問答
* 即時答題回饋
* 海龜知識解析

### 核心五：海上求生與救難

* 離岸流自救資訊
* 118 海巡署專線
* Intent(ACTION_DIAL)
