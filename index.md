# 海龜保育教育與海上安全通報 APP

## 專題介紹

本專題開發一套整合海龜保育教育與海上安全資訊之 Android App，提供賞龜熱點地圖、海龜品種介紹、保育法律與檢舉指南、海龜知識挑戰以及海上求生與救難資訊，提升民眾海洋保育與安全意識。

---

## 研究動機

* 民眾缺乏海龜品種與保育知識
* 不熟悉《野生動物保育法》，容易誤觸法規
* 親水活動中存在溺水與離岸流安全風險
* 缺少整合教育與安全資訊的行動應用程式

---

## 系統功能

### 核心一：全台賞龜地圖

透過 Google Maps API 顯示預設賞龜熱點座標，讓使用者查看小琉球、澎湖、綠島等海龜熱點位置資訊。

### 核心二：海龜品種大點名

串接 GBIF 開放資料 API，解析海龜觀測紀錄與品種資訊，並透過 RecyclerView 呈現資料。

### 核心三：保育法律與檢舉指南

整理《野生動物保育法》相關條文與罰則，提供保育法規與檢舉流程說明。

### 核心四：海龜守護者挑戰賽

以 RadioGroup 與 if-else 判斷機制建立互動式問答，提供答題回饋與知識解析。

### 核心五：海上求生與救難

提供離岸流自救資訊，並透過 Intent(ACTION_DIAL) 預填海巡署 118 專線，協助使用者快速進入撥號畫面。

## 系統架構

本系統以 MainActivity 作為首頁 Dashboard，整合五大核心功能模組，並透過 Java Controller 處理資料與功能邏輯。
